# Skribi

> *Skribi* signifie "écrire" en espéranto.

Skribi est un éditeur de texte riche natif, de type Word/Pages, 
entièrement construit en Rust avec [iced](https://github.com/iced-rs/iced).

## Statut

- En développement actif — pas encore utilisable.

## Objectifs

- Éditeur WYSIWYG natif (sans WebView)
- Paragraphes, titres, styles de texte
- Export HTML et .docx
- Interface native sur macOS, Windows et Linux

## Stack technique

- **Rust** — langage principal
- **iced** — framework UI natif (rendu GPU via wgpu)
